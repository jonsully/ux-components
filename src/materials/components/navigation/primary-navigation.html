--- notes: | ### Primary Navigation #### Active States Hovering over a navigation link will apply a hover state to the navigation item in order to indicate interactivity. location indicators on the navigation help people orient themselves on the site, especially if they are deeper within the website. This clear visual indicator can indicate which section someone is in. #### Subpages and Landing Pages Either all navigational elements should display their subpage links, or none of them; displaying subpages for only some sections may confuse people into thinking that sections that don’t display subpages actually do not have subpages. And if subpages aren’t listed in the primary navigation, ensure that secondary navigation is used consistently across all sections. ---
